C++ AUTO KEYWORD

Its many times that you get to see this auto keyword but you dont know what it means.Today I will show you
the meaning of it.
It is generally used when we dont know the datatype of ' the initilaizer for example:
Example
//Cpp auto example
//Notice we dont know the type s.begin() will provide so we use auto keyword and to print it we use a reference using a pointer
//Very important that the value is determined by the initilaizer

#include bits/stdc++.h

using namespace std;
int main(){
string s = "Monkey";
auto it = s.begin();

}
